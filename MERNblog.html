Overview of my MERN Stack App "Back Track"

This is a shared calendar MERN stack application which uses Mongo DB as a database Express Routing a React JS Fornt End and Node JS framework.

Creating a Topic

Each page is broken into components. The Make Topic page contians a table with the existing topics, a modal used to fill out informations for new topics, a loading icon and a footer that contains the add student button that triggers the modal. These components interact with a state that contians a topics array as well as a isFetching and isLoading boolean.
