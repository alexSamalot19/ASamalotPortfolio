Object Oriented Programming After a Few Years Writing Functional Code

4 comments Webdesign

I signed up for a Full Stack bootcamp to combine a few interests of mine: visual design, data analytics and modern coding practices. Web applications connect the user interface with the server and database using software and hardware that is constantly evolving. Object Oriented Programming (OOP) is a style of coding essential to web design and unlike coding languages it has remained a staple of this field. Increasing my use of this style of programming has been an essential development as someone who writes code.

My first exposure to software development was my B.S. Thesis where I ran a simple time series algorithm. I have been hooked ever since. At the time I thought it would be a stand alone project. I had no idea at the time I would be adapting the methodology through an M.S. Thesis and a publication. By the end I was guilty of separating data and functions with heavy reuse of similar interrelated functions with quite a bit of interdependency.

OOP takes out the uneccessary interdependency with four major principles, Encapsulation Abstraction, Inheritance and Polymorphism.

OOP combines a group of related variables and functions into objects with properites and methods.
